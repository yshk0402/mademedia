# Goal
note記事の無料パート（創業ストーリー）の
「一番面白い数字/事実/葛藤」を抽出し、
サムネイル（記事OGP）向けに
強度のあるコピー案を5つ生成する。

# Copy Strategy
・嘘・誇張・断言禁止（Fact only）
・「課題」or「衝撃数字」or「人の愚かさ」が主題
・認知バイアスを利用（損失回避/社会的証明）
・簡潔：15字〜25字
・ひと目で意味がわかる（抽象禁止）
・金の匂いは「根拠ある金」だけ採用
・英語の引用は極力避ける（視認性低下）
・入力はnote無料パートのみ。有料パートや過去案件の内容は使用禁止

# Structure
5案をそれぞれ異なるアプローチで：

1：数字フック（売上/成長率/資金）
2：失敗フック（落ちた/詰んだ/諦めた）
3：逆説フック（本来こうじゃない）
4：共感フック（あるある）
5：未来フック（これから起こる）

# Output Format (JSON)
{

    {"pattern": "failure", "text": "…"},
    {"pattern": "reverse", "text": "…"},
    {"pattern": "empathy", "text": "…"},
    {"pattern": "future", "text": "…"}
  ]
}

# Forbidden
・釣り（clickbait）
・感情を煽る言葉（神/革命/バズる等）
・確証なき推測
