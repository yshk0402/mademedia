# Goal
note（無料パート）の内容を、
X（旧Twitter）で最大拡散を狙える形式に最適化した
スレッド（8〜12ポスト）を作成する。

# Target Reader
・海外SaaSに興味あるが英語記事を追えない人
・副業/個人開発に興味がある層（25〜35歳）
・「今すぐ実務で使える情報」を好む層

# Strategy
・専門用語は説明を添えて平易化
・数値・具体例を最優先（抽象排除）
・1投稿に1メッセージ（AND禁止）
・「この続きは？」の余白を意図的に設計
・引用RTを誘う問いかけを1つ入れる
・広告/勧誘禁止（URLは末尾のみ）
・入力はnote無料パートのみ。有料パートの事例・数字・見出しの転用禁止。過去案件の生成物は参照しない

# Structure
01: 興味を一撃で奪う導入（最大50字）
02: サービス名と要点（数字1つ）
03〜08: ストーリー/課題/解決/証拠（1点ずつ）
09: 「なぜ日本で知られてない？」分析
10: 実用例（CONCRETE！）
11: 行動問いかけ（社会性・学び）
12: note誘導（URL）

# Tone
・友達に教える感じ
・断言せず、事実に基づく確信
・ひらがな多め、句読点少なめ、箇条書き活用

# Output Format (JSON)
{
  "thread": [
    {"order": 1, "text": "…"},
    {"order": 2, "text": "…"}
  ],
  "cta": {
    "note_url_placeholder": "<NOTE_URL>"
  }
}

# Forbidden
・最強/神/革命など煽りワード
・「使ってみて！」「登録して！」禁止
・未確認情報の断言
