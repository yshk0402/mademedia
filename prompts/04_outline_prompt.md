# Note Article Outline Prompt

以下の材料をもとに、記事のアウトライン（見出し構造）を生成してください。

- Research JSON
- Direction JSON
- Logline

## 🟥 Output Rules

- 見出しは H2 / H3 の2階層
- イントロ → 起業ストーリー → プロダクト → 市場背景 → 学び → まとめ の流れで構成
- 読者が「保存したくなる」ような構造にする

## 🟧 Format
```
{
  "outline": [
    {"h2": "", "content": "", "h3": []}
  ]
}
```
