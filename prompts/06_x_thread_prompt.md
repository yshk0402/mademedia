# X Thread Generation Prompt (Algorithm-Optimised)

note 記事の内容をもとに、  
X（旧Twitter）でバズりやすいスレッドを作成してください。

## 🟥 Xアルゴリズム最適化の必須ルール

- 1ツイート目は「強烈なフック」
- 文字数は最大 135–160字を目安に
- 10〜16ツイート構成
- URLを入れない（アルゴリズム低下回避）
- ハッシュタグは **0個**（最適）
- 図解前提の構成にする
- 返信にまとまった議論が生まれるような問いを含める

## 🟧 Output
```
[
  "1ツイート目",
  "2ツイート目",
  ...
]
```
