# Goal
読者が「すぐ使える」知識として持ち帰れる記事を生成する。
無料：創業ストーリーと価値の理解（3,000字）
有料：実務レベル活用方法（3,000字）

# Content Rules
・H2/H3は04/05に完全準拠（改変・追加禁止、順序入れ替え禁止）
　└04.headings.text を順番に H2/H3 として使用し、`paid_start_id` 直前に `<!-- FREE END -->`、直後に `<!-- PAID START -->` を挿入する
　└05.sections.body_draft の内容を本文化する。対応する heading id を必ず一致させる
・引用/数値は出典明記（脚注式）
・実例 > 解説 > 感想（意見禁止）
・広告/誘導表現禁止（公式紹介は脚注へ）
・不明情報は隠さず「現時点で未確認」と記述
・ユーザー行動につながる「次の一手」を1つ以上
・謝辞/締めは不要（情報価値優先）
・過去案件の生成物・メモは参照禁止。Target Service 以外の情報を使わない

# Structure
- 冒頭：本文導入300-500字
- FREE PART (H2/H3)
<!-- FREE END -->
<!-- PAID START -->
- PAID PART (H2/H3)
- まとめ：重要ポイントのリストアップのみ

# Tone
・平易（IQ100で理解）
・敬語／断定しすぎない科学的な文体

# Output
Markdown全文
